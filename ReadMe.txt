Antes de executar o código deve ter na mesma pasta os 4 datasets apresentados pelo professor. Deve também possuir o ficheiro py do primeiro trabalho para poder usar funções desse ficheiro.

Após isso deve dar RUN em todas as células onde irá ser apresentado um "menu". Nesse menu poderá selecionar se pretende divisão ou não de dados. Caso queira pode ainda selecionar o tamanho dos dados de teste. De seguida deve selecionar o dataset que pretende analisar.

IMPORTANTE: Caso possua um dataset novo, deve inserir o nome do ficheiro no comentário da linha 7 e apagar os "#" das linhas 7 e 8. 

Se pretender prever um dataset, na célula "DATASET" deve inserir o nome do ficheiro e correr a célula.
